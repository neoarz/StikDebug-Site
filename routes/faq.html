FAQ
  • Do I need WiFi?
    Yes – WiFi or Airplane Mode is required.
  • Is it open source?
    Yes – view on GitHub.
  • Supported iOS?
    iOS 17.4 and up.
  • Older iOS?
    No, update required.
Instructions
How to Install StikDebug (with AltStore PAL)
  1. Get AltStore PAL installed from altstore.io.
  2. Add the StikDebug source: Add
  3. Install via AltStore PAL – you're done!
Pairing File Instructions
  1. Create a pairing file using the SideStore guide.
  2. Zip and send it to your device via AirDrop, email, or Proton Drive.
How to Use StikDebug
  1. Open the app and click "Continue" to enable the loopback VPN.
  2. Tap "Connect" and select an app to debug.
How to View the Logs

Tap the Open Console button.
